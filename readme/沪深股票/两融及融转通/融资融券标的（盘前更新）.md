# 融资融券标的（盘前更新）

## 接口：margin_secs

### 描述
获取沪深京三大交易所融资融券标的（包括 ETF），每天盘前更新。

### 限量
单次最大 6000 行数据，可根据股票代码、交易日期、交易所代码循环提取。

### 积分
- 2000 积分可调取
- 5000 积分无总量限制
- 积分越高权限越大，具体参考权限说明。

---

## 输入参数

| 名称        | 类型 | 必选 | 描述 |
|------------|------|------|------|
| ts_code    | str  | N    | 标的代码 |
| trade_date | str  | N    | 交易日 |
| exchange   | str  | N    | 交易所（SSE 上交所，SZSE 深交所，BSE 北交所） |
| start_date | str  | N    | 开始日期 |
| end_date   | str  | N    | 结束日期 |

---

## 输出参数

| 名称        | 类型  | 默认显示 | 描述 |
|------------|------|--------|------|
| trade_date | str  | Y      | 交易日期 |
| ts_code    | str  | Y      | 标的代码 |
| name       | str  | Y      | 标的名称 |
| exchange   | str  | Y      | 交易所 |

---

## 接口用法

```python
pro = ts.pro_api()

# 获取 2024 年 4 月 17 日上交所融资融券标的
df = pro.margin_secs(trade_date='20240417', exchange='SSE')
```
