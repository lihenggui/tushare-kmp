getTradeCal

abstract suspend fun getTradeCal(params: TradeCalParams): List<TradeCalResult>

获取交易日历数据

获取股票市场的交易日历信息，包括交易日、非交易日的标识。 用于判断某个日期是否为交易日。

数据更新: 每年年初更新全年数据 数据权重: 免费

Return

交易日历数据列表

// 获取2024年交易日历
val calendar = tuShare.getTradeCal(
    TradeCalParams(startDate = "20240101", endDate = "20241231")
)

// 获取指定交易所日历
val sseCalendar = tuShare.getTradeCal(
    TradeCalParams(exchange = "SSE", startDate = "20240101")
)
