RetryStrategy

class RetryStrategy(val maxRetries: Int = 3, val base: Double = 2.0, val maxDelay: Duration = 60.seconds)

API请求重试策略配置

当API请求失败时，客户端会根据此策略进行重试。 支持指数退避算法，确保重试间隔逐渐增长。

Parameters

maxRetries

最大重试次数，超过此次数将抛出异常

base

退避算法的基数，用于计算重试间隔

maxDelay

最大重试延迟时间，防止重试间隔过长

// 基本重试策略：最多重试3次
val basicRetry = RetryStrategy(maxRetries = 3)

// 高级重试策略：自定义退避参数
val advancedRetry = RetryStrategy(
    maxRetries = 5,
    base = 1.5,
    maxDelay = 30.seconds
)
