# Sample App

JVM, JS and Native sample apps.

### Running the apps

| Target | command                                               |
|--------|-------------------------------------------------------|
| jvm    | `./gradlew :sample:jvm:run`                           |
| js     | `./gradlew :sample:js:jsNodeDevelopmentRun`           |
| native | `./gradlew :sample:native:runReleaseExecutableNative` |

_PS: environment variable `TUSHARE_` must be set._
